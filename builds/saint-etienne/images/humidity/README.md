# Photos problèmes d'humidité

Placez vos photos dans ce dossier avec les noms suivants :

## Noms de fichiers requis :

1. **remontees-capillaires.jpg** (ou .png)
   - Photo montrant des remontées capillaires
   - Traces d'humidité sur les murs, salpêtre, dégradation des enduits

2. **moisissures.jpg** (ou .png)
   - Photo montrant des moisissures
   - Taches noires, odeurs persistantes, dégradation des surfaces

3. **infiltrations.jpg** (ou .png)
   - Photo montrant des infiltrations d'eau
   - Taches au plafond, murs humides, dégâts des eaux

4. **condensation.jpg** (ou .png)
   - Photo montrant de la condensation
   - Buée sur les vitres, murs froids, moisissures dans les coins

## Format recommandé :
- Format : JPG ou PNG
- Résolution : 1200x900px minimum (ratio 4:3)
- Poids : < 500KB par image pour optimiser le chargement

## Note :
Les images seront automatiquement affichées sur le site dès qu'elles seront ajoutées dans ce dossier.
